이 블로그는

Hexo로 만들었습니다.

HEXO를 사용하면

깃허브 블로그를 간단하게 구축할 수 있습니다.

HEXO를 사용한 블로그 만들기

  1. 우선 Nodejs를 설치합니다. 사용할 적당한 개발툴도 있으면 좋습니다. 저는 VSC로 진행하였습니다.

  2. 깃허브에서 새로운 리포지토리를 생성합니다. -> 오너명.github.io

  3. npm install -g hexo 을 실행합니다.

  4. 블로그 프로젝트를 생성할 폴더를 선택,생성하여 개발툴의 터미널로 해당 폴더로 이동합니다.

  5. hexo init 블로그명으로 초기화합니다.

  6. 5.에서 만든 폴더로 이동하여 npm install 을 실행합니다.

  7. hexo server 를 실행하면 로컬에서 서버가 실행되어 확인할 수 있습니다.
    컨트롤 + c 키로 종료해주세요.

  8. 주요 설정파일은 _config.yml에 적혀있습니다.
    이 파일의 내용중에 아래 내용을 수정해봅니다.

title: 블로그제목

subtitle: 블로그부제목

author: 임의

language: ko

timezone: Asia/Seoul

url: https://오너명.github.io

그리고 가장 아래쪽의 deploy:를 수정합니다.

type: git

repo: 2.에서 만든 리포지토리의 ssh주소

branch: master

  1. npm install hexo-deployer-git –save 를 실행합니다.

  2. hexo deploy -g 를 실행합니다.
    이것으로 리포지토리에 자동 push 됩니다.

  3. 새로운 포스트 쓰기는 hexo new 타이틀명
